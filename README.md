# Civ5MapImage

This program will convert a Civ 5 map with the file extension .Civ5Map to a PNG image.

### Command-Line Usage

```
./Civ5MapImage.exe -input=[input filename] -output=[output filename]
```

Example
```
./Civ5MapImage.exe -input=earth.Civ5Map -output=earth.png
```
